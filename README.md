# PixelLab Test

PixelLab APIを使用して、ピクセルアートの草原を生成するシンプルなPythonスクリプトナリ！

## セットアップ

1. リポジトリをクローンするナリ：
```bash
git clone https://github.com/yourusername/pixellabtest.git
cd pixellabtest
```

2. 必要なパッケージをインストールするナリ：
```bash
uv pip install requests python-dotenv
```

3. `.env`ファイルを作成して、PixelLab APIキーを設定するナリ：
```bash
PIXELLAB_API_KEY=your_api_key_here
```

## 使い方

スクリプトを実行するナリ：
```bash
python pixel.py
```

成功すると、`grass_field.png`という名前のピクセルアート画像が生成されるナリ！
